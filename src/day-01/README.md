What i did /learned today:
I installed Rust according to the book, that included cargo and rustc,
i created the "Hello world" main.rs file using the fn function declaration.
Then i did the daily challange which had me writing five lines of code using "println!" inside the fn main() {}.
If you use the "!" after println you'll call a macro, to call a function remove the "!" at the end.
There are different ways to compile your code, one way is trough rustc [name].rs, the other ways are trough cargo run and cargo build where build, the difference is that build only creates an executable, while run compiles it and runs it so that you see the output.
You can also just check if it complies using cargo check without creating an executable.
Use cargo new to create a new project, cargo new --name <name> <path>.
